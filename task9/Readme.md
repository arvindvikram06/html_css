## Complex Responsive Layout with Grid and Flexbox

## Description

This project is a responsive web layout built using **HTML and CSS** that combines Grid and Flexbox. It includes a sticky navigation bar, sidebar, dynamic content layout, and overlapping UI element.

## Features

- Used **CSS Grid** to structure the main layout with a sidebar and content section.
- Implemented **Flexbox** for aligning and wrapping card components.
- Created a **sticky navigation bar and sidebar** using `position: sticky`.
- Designed an **inner grid system** inside the main section for better content organization.
- Implemented **responsive design** using media queries to adjust layout for smaller screens.
- Added **dynamic reordering** of content inside the main section for smaller screens.
- Built **overlapping elements** using `position`, `z-index`, and negative margins.
